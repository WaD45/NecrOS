Invoquer NecrOS

Installation rapide
# Sur un Alpine Linux fraîchement installé :

root@alpine:~# wget -qO- https://raw.githubusercontent.com/WaD45/NecrOS/main/install.sh | sh

# Ou manuellement :
root@alpine:~# git clone https://github.com/WaD45/NecrOS.git
root@alpine:~# cd NecrOS
root@alpine:~# sh necro_install.sh

# Options :
root@alpine:~# sh necro_install.sh --minimal   # Core seulement
root@alpine:~# sh necro_install.sh --full      # Tout + toolboxes
root@alpine:~# sh necro_install.sh --no-gui    # Sans interface
  1. Installer Alpine Linux
    Télécharger l'ISO depuis alpinelinux.org. Booter, login root (pas de mot de passe), lancer setup-alpine (mode "sys").
  2. Lancer l'installeur NecrOS
    L'installeur détecte automatiquement l'architecture, la RAM disponible, et adapte l'installation. Swap automatique sur les machines < 512 MB.
  3. Installer les toolboxes
    Lancez necros-toolbox et choisissez uniquement les outils dont vous avez besoin. Chaque toolbox est indépendante.
  4. Entrer dans l'abysse
    startx pour l'interface i3wm, ou restez en terminal. NecrOS est prêt.
